## Meta

### Using This Document

**When to Update:**
- After making any significant decision
- When learning an important lesson
- When key context changes
- During periodic reviews (weekly/monthly)

**How to Organize:**
- Keep active decisions at top
- Move old decisions to "Deprecated" section
- Use tags (#tag) for cross-referencing
- Link related decisions together

**Integration with Database:**
- Major entries are also stored in SQLite `memory_entries` table
- This file is the human-readable source of truth
- Database enables semantic search and retrieval
